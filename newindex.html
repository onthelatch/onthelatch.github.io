Setup Guide

For basic scanning, you only need a Spektrix¬Æ account with Scanning User permissions. To unlock the full feature set (attendee lists, scan reports, group scanning), follow these steps in your Spektrix¬Æ system:

1. User Permissions

Ensure your user account also has Collect Report permissions. This is set under Settings > Users > User Accounts.

2. Report Configuration

In Insights & Mailings, create a new Sales report named exactly 'Ticket Scanning Report'. Set the criteria to find tickets with a Seat Status of 'Sold' or 'Scanned', for events where the Start Date is 'Today'.

3. Report Schedule & Access Notes

Create a Report Schedule for this report, set to 'Run on demand', delivered as a Link in CSV format to the email addresses of your scanner users. For Access Notes, create a Ticket Attribute in Settings called 'Accessibility Needs'.
